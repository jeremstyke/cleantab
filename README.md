CleanTab cleans your Chrome browser in a single click — or automatically, on a schedule you choose. Pick what to delete, set the time range, and it's done in a second.

## Features

- **Manual / Automatic tabs** — clear separation between one-click cleaning and scheduled cleaning, each with its own independent settings
- **One-click cleaning** — no unnecessary confirmation
- **Automatic cleaning** — on startup, daily, weekly or monthly, with its own selection
- **Live countdown** — see exactly when the next automatic clean will run
- **Protected sites** — keep your sessions on important sites with the whitelist
- **Time range** — 1 hour, 24 hours, 7 days, 30 days, or everything
- **Desktop notifications** — get notified with the exact time when an automatic clean runs
- **Statistics & badges** — track your cleans, total space freed, and earn badges (Bronze to Legend)
- **Satisfying animation** — confetti, sounds and badges on every clean
- **Tab Rush mini-game** — a fun built-in game: close as many tabs as fast as you can! Two modes (Time Attack & Sprint)
- **7-day activity chart** — visualize your cleaning activity over the last week
- **Welcome page** — a friendly onboarding page on first install, with a pin tip
- **What's new page** — automatically shows new features after each update
- **Cleaning reminders** — a gentle nudge if your browser hasn't been cleaned in a while
- **Keyboard shortcut** — instant clean with Ctrl+Shift+E
- **Safety confirmation** before deleting saved passwords
- **Light & dark theme** — interface available in 22 languages
- **Private** — no data ever leaves your browser

## What you can clean

| Category | Description |
|----------|-------------|
| Cache | Cached images, scripts and files |
| Cookies | Session and login data |
| History | Visited pages |
| Local storage | localStorage, IndexedDB |
| Passwords | Saved credentials (with confirmation) |
