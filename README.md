## 🇬🇧 English

CleanTab cleans your Chrome browser in a single click — or automatically, on a schedule you choose. Pick what to delete, set the time range, and it's done in a second.

### Features

- **Manual / Automatic tabs** — clear separation between one-click and scheduled cleaning
- **One-click cleaning** — no unnecessary confirmation
- **Automatic cleaning** — on startup, daily, weekly or monthly
- **Live countdown** — see when the next automatic clean will run
- **Protected sites** — keep your sessions with the whitelist
- **Time range** — 1 hour, 24 hours, 7 days, 30 days, or everything
- **Desktop notifications** — get notified when an automatic clean runs
- **Statistics & badges** — track cleans, space freed, earn badges
- **Satisfying animation** — confetti, sounds and badges on every clean
- **Tab Rush mini-game** — close as many tabs as fast as you can!
- **7-day activity chart** — visualize your cleaning activity
- **Welcome page** — a friendly onboarding on first install
- **What's new page** — shows new features after each update
- **Cleaning reminders** — a gentle nudge if not cleaned in a while
- **Keyboard shortcut** — instant clean with Ctrl+Shift+E
- **Light & dark theme** — available in 22 languages
- **Private** — no data ever leaves your browser

### What you can clean

| Category | Description |
|---|---|
| Cache | Cached images, scripts and files |
| Cookies | Session and login data |
| History | Visited pages |
| Local storage | localStorage, IndexedDB |
| Passwords | Saved credentials (with confirmation) |

### Supported languages

Available in **22 languages**, auto-detected from your browser.

### Privacy

CleanTab does not collect, store, or transmit any personal data. Everything happens locally in your browser.

### Support

CleanTab is free, privacy-focused, and developed by a solo indie developer. Support it on [Ko-fi](https://ko-fi.com/cleantab)!

> CleanTab is an official partner of [DeleteMe](https://www.de33watrk.com/WCKMXS/KMKS9/), a service that removes your personal data from the web. Some banners in the extension link to DeleteMe as an affiliate.

- **File Manual / Automat** — separare clară între curățarea dintr-un clic și cea programată
- **Curățare dintr-un clic** — fără confirmare inutilă
- **Curățare automată** — la pornire, zilnic, săptămânal sau lunar
- **Numărătoare inversă live** — vezi următoarea curățare automată
- **Site-uri protejate** — păstrează sesiunile cu lista albă
- **Perioadă** — 1 oră, 24 ore, 7 zile, 30 zile, sau tot
- **Notificări** — fii anunțat la o curățare automată
- **Statistici și insigne** — urmărește curățările, câștigă insigne
- **Animație satisfăcătoare** — confetti, sunete și insigne
- **Mini-joc Tab Rush** — închide cât mai multe file cât mai repede!
- **Grafic de activitate 7 zile** — vizualizează activitatea
- **Pagină de bun venit** — la prima instalare
- **Pagină de noutăți** — arată noutățile după fiecare actualizare
- **Memento-uri** — un semnal dacă n-ai curățat de mult
- **Scurtătură de tastatură** — curățare instantanee cu Ctrl+Shift+E
- **Temă deschisă și întunecată** — disponibil în 22 de limbi
- **Privat** — niciun fel de date nu părăsesc browserul

### Ce poți curăța

| Categorie | Descriere |
|---|---|
| Cache | Imagini, scripturi și fișiere din cache |
| Cookie-uri | Date de sesiune și autentificare |
| Istoric | Pagini vizitate |
| Stocare locală | localStorage, IndexedDB |
| Parole | Credențiale salvate (cu confirmare) |

### Limbi acceptate

Disponibil în **22 de limbi**, detectate automat din browser.

### Confidențialitate

CleanTab nu colectează, stochează sau transmite date personale. Totul se întâmplă local în browserul tău.
